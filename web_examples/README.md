# Web Development Example Files

This directory contains complete, runnable web development examples from the Code Club sessions.

## How to Use

1. Each file corresponds to a session and demonstrates the concepts taught
2. Open HTML files directly in your web browser (double-click the file)
3. For React examples, you'll need to set up a React app first

## File Organisation

### HTML/CSS/JavaScript Examples (Sessions 61-72)

- **Session 61**: Introduction to Web Development
  - `session61_my_first_webpage.html` - Your first HTML page

- **Session 62**: HTML Basics
  - `session62_favourite_things.html` - HTML elements example

- **Session 63**: Introduction to CSS
  - `session63_styled_webpage.html` - Styled webpage with CSS

- **Session 65**: Introduction to JavaScript
  - `session65_javascript_interactive.html` - Interactive JavaScript page

- **Session 66**: JavaScript Variables
  - `session66_javascript_variables.html` - Variables demo

- **Session 67**: JavaScript Functions
  - `session67_function_calculator.html` - Calculator using functions

- **Session 68**: JavaScript DOM Manipulation
  - `session68_dom_manipulation.html` - DOM manipulation example

- **Session 70**: JavaScript Forms
  - `session70_form_validation.html` - Form with validation

- **Session 71**: JavaScript Arrays and Loops
  - `session71_array_manager.html` - Array management program

### React Examples (Sessions 73-84)

React examples require a React app to be set up. See individual session files for setup instructions.

## Running HTML Files

Simply double-click any `.html` file to open it in your web browser!

## Running React Examples

1. Make sure Node.js is installed
2. Create a new React app: `npx create-react-app my-app`
3. Copy the React code from session files into your app
4. Run: `npm start`

## Notes

- All examples use British English spelling
- Examples include comments to help you understand the code
- Feel free to modify and experiment with these examples!
- Some examples create files or use browser features - check the code for details

## Browser Compatibility

- All examples work in modern browsers (Chrome, Firefox, Edge, Safari)
- Some features may require a recent browser version
- React examples require Node.js and npm
