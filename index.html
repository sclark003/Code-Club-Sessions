BBC micro:bit Programming for SEN Students

Welcome to Code Club!

This portfolio contains hour-long coding sessions designed to introduce you to programming with BBC micro:bits, Cutebot robots, Arcade game development, and Python programming. Each session focuses on different features and coding concepts, building your skills week by week.

Start with Sessions 1-12 to learn micro:bit basics, then try Sessions 13-24 for Cutebot robotics (basics and advanced), Sessions 25-30 for Arcade game development, Sessions 31-36 for advanced game development, or Sessions 37-60 for Python programming!
