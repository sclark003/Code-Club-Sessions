BBC micro:bit Programming for SEN Students

Welcome to Code Club!

This portfolio contains hour-long coding sessions designed to introduce you to programming with BBC micro:bits, Cutebot robots, and Arcade game development. Each session focuses on different features and coding concepts, building your skills week by week.

Start with Sessions 1-12 to learn micro:bit basics, then try Sessions 13-18 for Cutebot robotics, Sessions 19-24 for Arcade game development, or Sessions 25-30 for advanced game development!
