# Python Example Programs

This directory contains complete, runnable Python programs from the Code Club sessions.

## How to Use

1. Each file corresponds to a session and demonstrates the concepts taught in that session
2. You can run any program by typing: `python filename.py`
3. Files ending with `_challenge.py` are example solutions to the challenge exercises

## File Organisation

## Notes

- All programs use British English spelling
- Programs include comments to help you understand the code
- Feel free to modify and experiment with these programs!
- Some programs create files (like `grades.json` or `grades.txt`) - these will be created in the same directory when you run the program

## Running Programs

To run a program, open a terminal/command prompt in this directory and type:

```bash
python session37_my_first_program.py
```

Or use Python IDLE:
1. Open IDLE
2. File ‚Üí Open
3. Select the .py file
4. Press F5 or Run ‚Üí Run Module
