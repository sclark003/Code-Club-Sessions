üöÄ Activity 1: What is CSS?

Step 1: CSS (Cascading Style Sheets) controls how your webpage looks - colours, fonts, sizes, spacing, and layout.
Step 2: Think of HTML as the structure (walls of a house) and CSS as the decoration (paint, furniture, layout)!

üíª Activity 2: Adding CSS to HTML

Step 1: There are three ways to add CSS:
  • Inline: Style directly in the HTML tag
  • Internal: Style in a <style> tag in the <head>
  • External: Style in a separate .css file (best practice!)
Step 2: Let's start with internal CSS (in the <head>): 
<head>
    <style>
        h1 {
            color: blue;
            font-size: 32px;
        }
        p {
            color: green;
        }
    </style>
</head>

üé® Activity 3: Basic CSS Properties

Step 1: Change text colour: 
h1 {
    color: blue;
}

p {
    color: #ff0000;  /* Red in hex code */
}
Step 2: Change font size: 
h1 {
    font-size: 32px;
}

p {
    font-size: 16px;
}
Step 3: Change font family: 
body {
    font-family: Arial, sans-serif;
}

h1 {
    font-family: "Times New Roman", serif;
}

Example: Here's a complete styled webpage: 
<!DOCTYPE html>
<html lang="en">

üí° Key Concepts

  • CSS: Cascading Style Sheets - controls appearance
  • Selector: The part that selects which element to style (like h1 or p)
  • Property: What you're changing (like color or font-size)
  • Value: The setting for the property (like blue or 32px)
  • Declaration: A property and value together (color: blue;)
  • Rule: A selector and its declarations
  • Class: A way to style multiple elements (using .classname)
