üì° What is an Ultrasonic Sensor?

Cutebot has an ultrasonic sensor that can measure distance! It sends out sound waves and measures how long they take to bounce back. This tells the robot how far away objects are. We can use this to make Cutebot avoid crashing into things!

The ultrasonic sensor measures distance in centimeters (cm). Typical range is about 2-400 cm.

üöÄ Activity 1: Read Distance

Step 1: Open MakeCode and create a new project called "Obstacle Avoidance"
Step 2: Make sure Cutebot extension is added
Step 3: Look in the Cutebot category for "ultrasonic distance" block
Step 4: Add "on button A pressed"
Step 5: Add "show number" and drag "ultrasonic distance" into it
Step 6: Test it - press button A and hold an object in front of Cutebot to see the distance!
The distance is measured in centimeters. Try holding objects at different distances to see the numbers change!

üé® Activity 2: Simple Obstacle Detection

Step 1: Create a "forever" loop
Step 2: Inside, add an "if" statement
Step 3: Check if distance is less than 10 cm (obstacle too close)
Step 4: If obstacle detected:
  • Stop moving
  • Move backward for 1 second
  • Turn right (or left)
  • Continue forward
Step 5: Otherwise, move forward
Step 6: Test it - Cutebot should stop and avoid objects!

üéØ Activity 3: Smart Obstacle Avoidance

Step 1: Improve the algorithm:
  • If distance < 5 cm: Emergency stop and reverse
  • Else if distance < 15 cm: Slow down and prepare to turn
  • Else if distance < 30 cm: Continue but be cautious
  • Else: Move forward at normal speed
Step 2: When avoiding, check which direction is clearer:
  • Turn left and check distance
  • Turn right and check distance
  • Go in the direction with more space
Step 3: Add visual feedback - show distance on micro:bit screen

üåü Activity 4: Display Warnings

Step 1: Add visual warnings on the micro:bit:
  • Distance > 30 cm: Show happy face (safe)
  • Distance 15-30 cm: Show neutral face (caution)
  • Distance < 15 cm: Show sad face (danger!)
Step 2: Add sound warnings - play a beep when obstacle is close
Step 3: Now you have both visual and audio feedback!
Create an advanced obstacle avoider:
  • Continuously scans for obstacles while moving
  • Shows distance on screen
  • Plays warning sounds when close to obstacles
  • Chooses the best direction to turn (left or right)
  • Can navigate around multiple obstacles
  • Button A to start/stop
  • Button B to toggle between cautious and normal speed

üí° Key Concepts

  • Ultrasonic Sensor: Uses sound waves to measure distance to objects
  • Distance Measurement: How far away something is (in centimeters)
  • Obstacle Detection: Finding objects in the robot's path
  • Collision Avoidance: Changing direction to avoid hitting things
  • Safety Zones: Different distances that trigger different behaviors
  • Autonomous Navigation: Robot making its own decisions to avoid obstacles
