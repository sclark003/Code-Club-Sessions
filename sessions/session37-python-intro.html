üöÄ Activity 1: What is Python?

Step 1: Python is a programming language that's easy to read and write. It's used for websites, games, data analysis, and much more!
Step 2: Python code is written in plain text files with a .py extension (like my_program.py)
Step 3: Python is known for being beginner-friendly because it reads almost like English!

üíª Activity 2: Installing Python

Step 1: Go to python.org/downloads
Step 2: Download the latest version of Python (Python 3.x)
Step 3: Run the installer. Make sure to check the box that says "Add Python to PATH" - this is important!
Step 4: Click "Install Now" and wait for it to finish
If you're using a school computer, Python might already be installed! Ask your teacher to check.

üé® Activity 3: Your First Python Program

Step 1: Open Python IDLE (search for "IDLE" in your Start menu or Applications folder)
Step 2: You'll see a window called "Python Shell" - this is where you can type Python code directly
Step 3: Type this and press Enter: 
print("Hello, World!")
Step 4: You should see "Hello, World!" appear on the next line! Congratulations - you've written your first Python program!
Step 5: Try changing the message: 
print("Hello, Code Club!")

üìù Activity 4: Creating a Python File

Step 1: In IDLE, go to File ‚Üí New File (or press Ctrl+N)
Step 2: Type your program: 
print("Welcome to Python!")
print("This is my first program.")
print("Python is fun!")
Step 3: Save the file (File ‚Üí Save) as my_first_program.py
Step 4: Press F5 or go to Run ‚Üí Run Module to run your program
Step 5: Watch the output appear in the Python Shell window!

üìã Complete Example Program

Example: Here's a complete program you can copy and run: 
# my_first_program.py
# This is my first Python program!

# Print a welcome message
print("Welcome to Python!")
print("=" * 30)

# Print some information
print("My name is Sarah")
print("I am learning Python")
print("Python is fun!")

# Print with different styles
print("\nHere are some things I can do:")
print("- Display text")
print("- Do calculations")
print("- Create programs")

print("\n" + "=" * 30)
print("End of program")
Output: When you run this program, you should see: 
Welcome to Python!
==============================
My name is Sarah
I am learning Python
Python is fun!

Here are some things I can do:
- Display text
- Do calculations
- Create programs

==============================
End of program

üí° Key Concepts

  • Python: A high-level programming language that's easy to learn and powerful
  • IDLE: Python's built-in development environment (Integrated Development and Learning Environment)
  • print(): A function that displays text on the screen
  • String: Text enclosed in quotes (single or double quotes both work)
  • .py file: A Python program file that contains your code
