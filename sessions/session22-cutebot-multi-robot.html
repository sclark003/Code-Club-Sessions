ü§ù Multi-Robot Systems

Today we'll learn to make multiple Cutebots work together! By coordinating robots, we can accomplish tasks that would be difficult for a single robot.

üöÄ Activity 1: Radio Setup for Robots

Step 1: Set up radio communication (review Session 10 if needed)
Step 2: Assign robot IDs:
  • Robot 1: ID = 1
  • Robot 2: ID = 2
  • Use same radio group for all robots
Step 3: Create message types:
  • "START" - Begin mission
  • "STOP" - Stop moving
  • "POSITION" - Share position
  • "STATUS" - Share status

üé® Activity 2: Leader-Follower System

Step 1: Designate one robot as leader, others as followers
Step 2: Leader robot:
  • Moves based on buttons or sensors
  • Sends position/status to followers
  • Makes decisions for the group
Step 3: Follower robots:
  • Receive messages from leader
  • Move to follow leader
  • Maintain distance from leader
Step 4: Test - followers should follow the leader!

üéØ Activity 3: Synchronized Movement

Step 1: Make robots move in sync
Step 2: One robot sends "GO" command
Step 3: All robots receive command and move together:
  • Move forward for 2 seconds
  • Turn left together
  • Move forward again
Step 4: Create a synchronized dance or pattern!

üåü Activity 4: Cooperative Task

Step 1: Design a task requiring cooperation:
  • Robot 1: Finds and follows line
  • Robot 2: Clears obstacles ahead
  • Both: Work together to complete mission
Step 2: Program communication:
  • Robot 1 sends "OBSTACLE" when it sees one
  • Robot 2 receives message and moves to clear it
  • Robot 2 sends "CLEAR" when done
  • Robot 1 continues when path is clear
Step 3: Test the cooperative mission!

üéÆ Activity 5: Formation Flying

Step 1: Create robots that move in formation
Step 2: Formation types:
  • Line: One behind the other
  • V-formation: Like birds flying
  • Square: Four robots in a square
Step 3: Use radio to coordinate positions
Step 4: Maintain formation while moving
Create a multi-robot system with:
  • At least 2 robots working together
  • Radio communication between robots
  • Leader-follower or cooperative behavior
  • Synchronized movements
  • Complete cooperative mission
  • Formation movement capability

üí° Key Concepts

  • Multi-Robot System: Multiple robots working together
  • Coordination: Making robots work in harmony
  • Leader-Follower: One robot leads, others follow
  • Synchronization: Robots doing things at the same time
  • Cooperation: Robots helping each other
  • Formation: Robots arranged in a pattern
  • Communication: Robots sharing information
