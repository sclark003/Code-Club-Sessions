üöÄ Activity 1: What is the DOM?

Step 1: DOM stands for Document Object Model - it's how JavaScript sees and interacts with your HTML.
Step 2: JavaScript can find elements and change them!

üíª Activity 2: Selecting Elements

Step 1: Select elements by ID, class, or tag: 
// By ID
let element = document.getElementById('myId');

// By class (returns array)
let elements = document.getElementsByClassName('myClass');

// By tag name
let paragraphs = document.getElementsByTagName('p');

// Modern way - querySelector
let element = document.querySelector('#myId');
let elements = document.querySelectorAll('.myClass');

üé® Activity 3: Changing Content

Step 1: Change text content: 
let heading = document.getElementById('myHeading');
heading.textContent = "New Text";
heading.innerHTML = "<strong>Bold Text</strong>";
Step 2: Change styles: 
let element = document.getElementById('myElement');
element.style.color = 'red';
element.style.fontSize = '24px';
element.style.backgroundColor = 'yellow';

üìã Complete Example Program

Example: Here's a complete DOM manipulation example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>DOM Manipulation</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            max-width: 600px;
            margin: 50px auto;
            padding: 20px;
        }
        #content {
            padding: 20px;
            border: 2px solid #333;
            margin: 20px 0;
            min-height: 100px;
        }
        button {
            padding: 10px 20px;
            margin: 5px;
            font-size: 16px;
            background-color: #0066cc;
            color: white;
            border: none;
            border-radius: 5px;
            cursor: pointer;
        }
        .highlight {
            background-color: yellow;
            padding: 5px;
        }
    </style>
</head>
<body>
    <h1 id="mainHeading">DOM Manipulation Demo</h1>
    <div id="content">This content can be changed!</div>
    
    <button onclick="changeText()">Change Text</button>
    <button onclick="changeColour()">Change Colour</button>
    <button onclick="addElement()">Add Element</button>
    <button onclick="removeElement()">Remove Element</button>
    <button onclick="reset()">Reset</button>
    
    <script>
        function changeText() {
            const content = document.getElementById('content');
            content.textContent = 'Text changed with JavaScript!';
        }
        
        function changeColour() {
            const content = document.getElementById('content');
            const colours = ['red', 'blue', 'green', 'purple', 'orange'];
            const randomColour = colours[Math.floor(Math.random() * colours.length)];
            content.style.backgroundColor = randomColour;
            content.style.color = 'white';
        }
        
        function addElement() {
            const content = document.getElementById('content');
            const newElement = document.createElement('p');
            newElement.textContent = 'New element added!';
            newElement.className = 'highlight';
            content.appendChild(newElement);
        }
        
        function removeElement() {
            const content = document.getElementById('content');
            if (content.children.length > 0) {
                content.removeChild(content.lastChild);
            }
        }
        
        function reset() {
            const content = document.getElementById('content');
            content.innerHTML = 'This content can be changed!';
            content.style.backgroundColor = '';
            content.style.color = '';
        }
    </script>
</body>
</html>
Create a program that:
  • Has a button that changes the page background colour
  • Adds a new list item when clicked
  • Changes the font size of headings
  • Shows/hides an element

üí° Key Concepts

  • DOM: Document Object Model - representation of HTML
  • getElementById(): Finds element by ID
  • querySelector(): Modern way to find elements
  • textContent: Gets/sets text content
  • innerHTML: Gets/sets HTML content
  • style: Object to change CSS styles
  • createElement(): Creates new HTML elements
  • appendChild(): Adds element to page
