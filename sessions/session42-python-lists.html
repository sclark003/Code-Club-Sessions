ğŸ’» Activity 2: Accessing List Items

Step 1: Like strings, you can access list items by index: 
fruits = ["apple", "banana", "orange", "grape"]

print(fruits[0])  # apple (first item)
print(fruits[1])  # banana
print(fruits[-1])  # grape (last item)
print(fruits[-2])  # orange
Step 2: You can also slice lists: 
fruits = ["apple", "banana", "orange", "grape", "kiwi"]
print(fruits[1:3])  # ['banana', 'orange']
print(fruits[:2])   # ['apple', 'banana']
print(fruits[2:])   # ['orange', 'grape', 'kiwi']

ğŸ¨ Activity 3: Modifying Lists

Step 1: You can change items in a list: 
fruits = ["apple", "banana", "orange"]
fruits[1] = "pear"
print(fruits)  # ['apple', 'pear', 'orange']
Step 2: Add items to a list: 
fruits = ["apple", "banana"]
fruits.append("orange")  # Add to the end
print(fruits)

fruits.insert(1, "grape")  # Insert at position 1
print(fruits)
Step 3: Remove items: 
fruits = ["apple", "banana", "orange"]
fruits.remove("banana")  # Remove by value
print(fruits)

fruits.pop()  # Remove last item
print(fruits)

fruits.pop(0)  # Remove item at index 0
print(fruits)

ğŸ’¡ Key Concepts

  • List: An ordered collection of items stored in square brackets []
  • Index: The position of an item in a list (starts at 0)
  • append(): Adds an item to the end of a list
  • insert(): Adds an item at a specific position
  • remove(): Removes an item by value
  • pop(): Removes an item by index (or last item if no index given)
  • sort(): Arranges list items in order
  • len(): Returns the number of items in a list
