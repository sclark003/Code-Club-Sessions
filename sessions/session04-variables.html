ğŸ’­ What is a Variable?

A variable is like a box where you can store information. You give the box a name, and you can put numbers (or text) inside it. Later, you can look inside the box to see what's there, or change what's inside!

ğŸš€ Activity 1: Create Your First Variable

Step 1: Open MakeCode and create a new project called "Variables"
Step 2: Go to the "Variables" category and click "Make a Variable"
Step 3: Name your variable "score" (or any name you like!)
Step 4: You'll see new blocks appear: "set score to 0" and "change score by 1"
Step 5: Drag "set score to 0" into the "on start" block
Step 6: Add an "on button A pressed" block
Step 7: Inside it, add "change score by 1"
Step 8: Add a "show number" block and drag the "score" variable into it (from Variables category)
Step 9: Test it! Press button A and watch the score increase!
Variables start at 0. Every time you press button A, the score increases by 1. The variable "remembers" the number!

Step 3: Add "show string" that says "Reset!"
Step 4: Now you have a counter: Button A increases, Button B resets!
