üî¨ Sensor Fusion

Today we'll learn to combine information from multiple sensors to make Cutebot smarter! By using several sensors together, we can create more reliable and intelligent behaviors.

üöÄ Activity 1: Line + Obstacle Detection

Step 1: Combine line following with obstacle avoidance
Step 2: Create a decision tree:
  • If obstacle detected: Stop and avoid (highest priority)
  • Else if line detected: Follow line
  • Else: Search for line or continue forward
Step 3: Use both sensors continuously in your main loop
Step 4: Test - Cutebot should follow lines while avoiding obstacles!

üé® Activity 2: Light-Seeking Robot

Step 1: Use micro:bit's light sensor to find the brightest area
Step 2: Combine with obstacle avoidance:
  • Move toward light
  • But avoid obstacles
  • If blocked, find another path toward light
Step 3: Use light level to determine direction:
  • Turn left and check light level
  • Turn right and check light level
  • Go in direction with more light
Step 4: Create a robot that seeks light while avoiding obstacles!

üéØ Activity 3: Temperature-Aware Navigation

Step 1: Use temperature sensor to detect hot areas
Step 2: Create behavior:
  • If temperature too high: Move away (safety)
  • If temperature normal: Continue normal navigation
  • Combine with other sensors for complete behavior
Step 3: Use temperature to make navigation decisions

üåü Activity 4: Multi-Sensor Decision Matrix

Step 1: Create a system that considers all sensors:
  • Line sensors: Where is the line?
  • Ultrasonic: Are there obstacles?
  • Light: Where is it bright?
  • Temperature: Is it safe?
Step 2: Assign priorities:
  • Priority 1: Safety (obstacles, temperature)
  • Priority 2: Navigation (line following)
  • Priority 3: Optimization (light seeking)
Step 3: Make decisions based on all sensor inputs

üéÆ Activity 5: Sensor Calibration

Step 1: Calibrate sensors at startup
Step 2: Measure baseline values:
  • Normal light level
  • Normal temperature
  • Normal distance readings
Step 3: Use these baselines to detect changes
Step 4: Adjust behavior based on calibrated values
Create an intelligent robot that:
  • Uses all available sensors (line, ultrasonic, light, temperature)
  • Makes smart decisions based on multiple sensor inputs
  • Has a priority system for conflicting information
  • Calibrates sensors at startup
  • Adapts behavior based on sensor readings
  • Can navigate complex environments using sensor fusion

üí° Key Concepts

  • Sensor Fusion: Combining data from multiple sensors
  • Decision Matrix: A system for making decisions based on multiple inputs
  • Priority System: Ranking importance of different factors
  • Calibration: Setting baseline values for sensors
  • Conflicting Information: When sensors give different signals
  • Multi-Sensor Behavior: Actions based on multiple sensor readings
