üöÄ Activity 1: What is Context?

Step 1: Context lets you share data across many components without passing props through every level.
Step 2: Useful for theme, user data, or settings that many components need!

üíª Activity 2: Creating Context

Step 1: Create and use context: 
import { createContext, useContext, useState } from 'react';

const ThemeContext = createContext();

function ThemeProvider({ children }) {
  const [theme, setTheme] = useState('light');
  
  return (
    <ThemeContext.Provider value={{ theme, setTheme }}>
      {children}
    </ThemeContext.Provider>
  );
}

function useTheme() {
  return useContext(ThemeContext);
}

üìã Complete Example Program

Example: Here's a complete theme context example: 
// ThemeContext.js
import { createContext, useContext, useState } from 'react';

const ThemeContext = createContext();

export function ThemeProvider({ children }) {
  const [theme, setTheme] = useState('light');
  
  const toggleTheme = () => {
    setTheme(theme === 'light' ? 'dark' : 'light');
  };
  
  return (
    <ThemeContext.Provider value={{ theme, toggleTheme }}>
      {children}
    </ThemeContext.Provider>
  );
}

export function useTheme() {
  const context = useContext(ThemeContext);
  if (!context) {
    throw new Error('useTheme must be used within ThemeProvider');
  }
  return context;
}

// App.js
import { ThemeProvider } from './ThemeContext';
import Header from './Header';
import Content from './Content';

function App() {
  return (
    <ThemeProvider>
      <div className="App">
        <Header />
        <Content />
      </div>
    </ThemeProvider>
  );
}

// Header.js
import { useTheme } from './ThemeContext';

function Header() {
  const { theme, toggleTheme } = useTheme();
  
  return (
    <header className={`header ${theme}`}>
      <h1>My App</h1>
      <button onClick={toggleTheme}>
        Switch to {theme === 'light' ? 'Dark' : 'Light'} Mode
      </button>
    </header>
  );
}

// Content.js
import { useTheme } from './ThemeContext';

function Content() {
  const { theme } = useTheme();
  
  return (
    <main className={`content ${theme}`}>
      <p>Current theme: {theme}</p>
      <p>This content changes colour based on the theme!</p>
    </main>
  );
}
Create a context for:
  • User information (name, email)
  • Multiple components that display user info
  • A component that can update user info
  • All components update when user info changes

üí° Key Concepts

  • Context: Way to share data across components
  • createContext(): Creates a new context
  • Provider: Wraps components to provide context value
  • useContext(): Hook to access context value
  • Prop drilling: Passing props through many levels (Context avoids this)
