ğŸ’» Activity 2: Formatting Dates

Step 1: Format dates in different ways: 
from datetime import datetime

now = datetime.now()

# Different formats
print(now.strftime("%Y-%m-%d"))           # 2024-12-25
print(now.strftime("%d/%m/%Y"))           # 25/12/2024
print(now.strftime("%A, %B %d, %Y"))      # Wednesday, December 25, 2024
print(now.strftime("%H:%M:%S"))           # 14:30:45
print(now.strftime("%I:%M %p"))           # 02:30 PM
Step 2: Common format codes:
  • %Y - Year (4 digits)
  • %m - Month (01-12)
  • %d - Day (01-31)
  • %H - Hour (00-23)
  • %M - Minute (00-59)
  • %A - Weekday name
  • %B - Month name

ğŸ¨ Activity 3: Date Arithmetic

Step 1: Calculate differences and add/subtract time: 
from datetime import date, timedelta

today = date.today()
birthday = date(2010, 5, 15)

# Calculate age
age = today - birthday
print(f"Days old: {age.days}")
print(f"Years old: {age.days // 365}")

# Add/subtract days
next_week = today + timedelta(days=7)
yesterday = today - timedelta(days=1)
print(f"Next week: {next_week}")
print(f"Yesterday: {yesterday}")

# Add weeks, hours, etc.
future = today + timedelta(weeks=2, days=3)
print(f"Future date: {future}")

ğŸ¯ Activity 5: Age Calculator

Step 1: Create an age calculator: 
from datetime import date

def calculate_age(birth_year, birth_month, birth_day):
    today = date.today()
    birthday = date(birth_year, birth_month, birth_day)
    age = today - birthday
    years = age.days // 365
    remaining_days = age.days % 365
    return years, remaining_days

birth_year = int(input("Enter birth year: "))
birth_month = int(input("Enter birth month: "))
birth_day = int(input("Enter birth day: "))

years, days = calculate_age(birth_year, birth_month, birth_day)
print(f"You are {years} years and {days} days old!")

Step 1: Calculate days until a future event: 
from datetime import date

def days_until_event(event_year, event_month, event_day):
    today = date.today()
    event_date = date(event_year, event_month, event_day)
    difference = event_date - today
    return difference.days

event_year = int(input("Event year: "))
event_month = int(input("Event month: "))
event_day = int(input("Event day: "))

days = days_until_event(event_year, event_month, event_day)

if days > 0:
    print(f"{days} days until the event!")
elif days == 0:
    print("The event is today!")
else:
    print(f"The event was {abs(days)} days ago.")
