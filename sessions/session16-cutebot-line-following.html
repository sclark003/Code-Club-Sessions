üõ§Ô∏è What is Line Following?

Cutebot has sensors on the bottom that can detect dark lines! By reading these sensors, we can make Cutebot automatically follow a black line on a light surface. This is how many robots navigate!

You'll need to create a track first! Use black electrical tape or a black marker to draw a line on white paper or a light floor.

üöÄ Activity 1: Read Line Sensors

Step 1: Open MakeCode and create a new project called "Line Following"
Step 2: Make sure Cutebot extension is added
Step 3: Look in the Cutebot category for sensor blocks
Step 4: Find "tracking left/right line sensor" blocks
Step 5: Add "on button A pressed" to test sensors
Step 6: Add "show number" and drag "tracking left line sensor" into it
Step 7: Test it - place Cutebot over a black line and press button A to see the sensor value!
Line sensors usually return 1 when they detect a line (dark) and 0 when they don't (light surface).

üé® Activity 2: Simple Line Following

Step 1: Create a "forever" loop
Step 2: Inside, add an "if" statement
Step 3: Check if left sensor detects a line:
  • If left sensor = 1 (detects line): Turn left slightly
  • Else if right sensor = 1 (detects line): Turn right slightly
  • Else: Move forward (both sensors on light surface)
Step 4: Add small pauses between checks
Step 5: Place Cutebot on your line track and watch it follow the line!

üéØ Activity 3: Improved Line Following

Step 1: Improve the algorithm:
  • If both sensors detect line: Move forward (on the line)
  • If left sensor detects line: Turn left (line is to the left)
  • If right sensor detects line: Turn right (line is to the right)
  • If neither detects line: Move forward slowly (might have lost the line)
Step 2: Adjust speeds - turning should be slower than forward movement
Step 3: Test on different track shapes - straight lines, curves, and corners!

üåü Activity 4: Display Sensor Status

Step 1: Add visual feedback on the micro:bit screen
Step 2: Show different icons based on what sensors detect:
  • Both sensors: Show arrow pointing forward
  • Left sensor: Show arrow pointing left
  • Right sensor: Show arrow pointing right
  • Neither: Show a question mark
Step 3: This helps you see what the robot is "thinking"!
Create an advanced line follower:
  • Follows the line smoothly
  • Shows sensor status on micro:bit display
  • Plays a sound when it loses the line
  • Button A to start/stop line following
  • Button B to adjust speed
  • Can handle sharp turns and curves!

üí° Key Concepts

  • Line Sensor: A sensor that detects dark lines on light surfaces
  • Sensor Reading: Getting a value (0 or 1) from the sensor
  • Autonomous: The robot works on its own without constant control
  • Feedback Loop: Reading sensors ‚Üí Making decision ‚Üí Moving ‚Üí Reading sensors again
  • Algorithm: A set of rules (like "if left sensor detects line, turn left")
