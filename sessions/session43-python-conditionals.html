üöÄ Activity 1: Simple if Statements

Step 1: An if statement checks a condition and runs code only if it's True: 
age = 15

if age >= 13:
    print("You are a teenager!")

print("This always runs")
Step 2: Notice the colon (:) and the indentation! The code inside the if statement must be indented.
In Python, indentation is very important! Use 4 spaces (or 1 tab) to indent code inside if statements.

üíª Activity 2: if/else Statements

Step 1: Use else to run code when the condition is False: 
age = int(input("How old are you? "))

if age >= 18:
    print("You are an adult!")
else:
    print("You are not yet an adult.")
Step 2: Try running this with different ages!

üé® Activity 3: Comparison Operators

Step 1: Python has several comparison operators: 
# Equal to
5 == 5  # True

# Not equal to
5 != 3  # True

# Less than
3 < 5   # True

# Greater than
5 > 3   # True

# Less than or equal to
5 <= 5  # True

# Greater than or equal to
5 >= 3  # True
Step 2: Try using these in if statements: 
score = int(input("What was your score? "))

if score >= 90:
    print("Excellent! Grade: A")
elif score >= 80:
    print("Good job! Grade: B")
elif score >= 70:
    print("Well done! Grade: C")
elif score >= 60:
    print("Keep trying! Grade: D")
else:
    print("You can do better! Grade: F")

üìä Activity 4: Multiple Conditions with elif

Step 1: Use elif (else if) to check multiple conditions: 
weather = input("What's the weather like? (sunny/rainy/cloudy): ")

if weather == "sunny":
    print("Great day for a picnic!")
elif weather == "rainy":
    print("Don't forget your umbrella!")
elif weather == "cloudy":
    print("Might rain later, bring a jacket.")
else:
    print("I'm not sure about that weather!")
Step 2: Create your own program with multiple conditions!

üéØ Activity 5: Logical Operators

Step 1: You can combine conditions using and, or, and not: 
age = int(input("How old are you? "))
has_license = input("Do you have a license? (yes/no): ")

if age >= 18 and has_license == "yes":
    print("You can drive!")
elif age >= 18 and has_license == "no":
    print("You're old enough but need a license.")
else:
    print("You're too young to drive.")

# Using 'or'
temperature = int(input("What's the temperature? "))
if temperature < 0 or temperature > 35:
    print("Extreme weather!")
else:
    print("Normal temperature.")

üåü Activity 6: Building a Decision Program

Step 1: Create a program that helps decide what to do: 
print("What should I do today?")
print("=" * 25)

weather = input("Is it sunny? (yes/no): ")
homework = input("Is homework done? (yes/no): ")

if weather == "yes" and homework == "yes":
    print("Great! Go outside and play!")
elif weather == "yes" and homework == "no":
    print("Finish homework first, then you can play!")
elif weather == "no" and homework == "yes":
    print("Perfect day to read a book or play games inside.")
else:
    print("Time to do homework!")
Create a program that:
  • Asks for a number
  • Tells if it's positive, negative, or zero
  • Tells if it's even or odd
  • Tells if it's a single digit or multiple digits

üí° Key Concepts

  • if statement: Runs code only if a condition is True
  • else: Runs code when the if condition is False
  • elif: Checks another condition if the previous ones were False
  • Comparison Operators: == (equal), != (not equal), <, >, <=, >=
  • Logical Operators: and, or, not
  • Indentation: Code inside if/else must be indented (4 spaces)
  • Condition: An expression that evaluates to True or False
