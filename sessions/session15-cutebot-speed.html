ğŸ¨ Activity 2: Change Speed with Buttons

Step 1: Add "on button B pressed" to increase speed
Step 2: Inside it, add "change speed by 10"
Step 3: Add "show number" to display the current speed
Step 4: Add "on shake" to decrease speed by 10
Step 5: Test it:
  • Press B to speed up (shows new speed)
  • Shake to slow down (shows new speed)
  • Press A to move at the current speed
You might want to add limits! Use if statements to make sure speed stays between 0 and 100.

ğŸ¯ Activity 3: Speed Limits

Step 1: Add an "if" block when changing speed
Step 2: Check if speed is less than 100 before increasing
Step 3: Check if speed is greater than 0 before decreasing
Step 4: Now speed can't go below 0 or above 100!

Step 1: Create a program that adjusts speed based on light level
Step 2: Add a "forever" loop
Step 3: Inside, set speed based on light level:
  • If light level > 150: Set speed to 80 (bright = fast)
  • Else if light level > 50: Set speed to 50 (medium = medium speed)
  • Else: Set speed to 20 (dark = slow)
Step 4: Make Cutebot move forward at that speed
Step 5: Shine a light on the micro:bit to make Cutebot speed up!
Create a smart speed system:
  • Button A increases speed by 5 (max 100)
  • Button B decreases speed by 5 (min 0)
  • Shake resets speed to 50
  • Show speed on micro:bit display
  • Tilt forward moves at current speed
  • Add a "turbo mode" that doubles the speed when both buttons are pressed!
