üöÄ Activity 1: String Concatenation

Step 1: You can join strings together using the + operator: 
first_name = "Alex"
last_name = "Smith"
full_name = first_name + " " + last_name
print(full_name)
Step 2: You can also repeat strings using *: 
cheer = "Hip " * 3 + "Hooray!"
print(cheer)

separator = "-" * 20
print(separator)

üíª Activity 2: String Indexing

Step 1: Each character in a string has a position (index). The first character is at position 0: 
word = "Python"
print(word[0])  # P
print(word[1])  # y
print(word[2])  # t
print(word[5])  # n
Step 2: You can also count from the end using negative numbers: 
word = "Python"
print(word[-1])  # n (last character)
print(word[-2])  # o (second to last)
Remember: Python starts counting at 0, not 1! The first character is at index 0.

üé® Activity 3: String Slicing

Step 1: You can get parts of a string using slicing: 
text = "Hello, World!"

# Get characters from position 0 to 5 (not including 5)
print(text[0:5])  # Hello

# Get from position 7 to the end
print(text[7:])  # World!

# Get from start to position 5
print(text[:5])  # Hello

# Get every other character
print(text[::2])  # Hlo ol!
Step 2: Try slicing your own strings!

üìä Activity 4: String Methods

Step 1: Strings have built-in methods (functions) you can use: 
text = "  Hello, Python!  "

# Convert to uppercase
print(text.upper())

# Convert to lowercase
print(text.lower())

# Remove whitespace from ends
print(text.strip())

# Replace text
print(text.replace("Python", "World"))

# Find position of text
print(text.find("Hello"))

# Count occurrences
print("Hello, Hello, World".count("Hello"))
Step 2: Try these methods with your own strings

üéØ Activity 5: String Formatting

Step 1: We've seen f-strings, but there are other ways to format: 
name = "Alex"
age = 12

# f-string (modern way)
print(f"My name is {name} and I am {age} years old.")

# .format() method
print("My name is {} and I am {} years old.".format(name, age))

# % formatting (older way)
print("My name is %s and I am %d years old." % (name, age))
Step 2: f-strings are usually the easiest and most readable!

üåü Activity 6: String Challenges

Create a program that:
  • Asks for a sentence
  • Counts how many words are in it (hint: use .split())
  • Finds the longest word
  • Replaces all spaces with dashes
  • Prints the sentence backwards

üí° Key Concepts

  • String: A sequence of characters (text)
  • Concatenation: Joining strings together with +
  • Index: The position of a character in a string (starts at 0)
  • Slicing: Getting part of a string using [start:end]
  • Method: A function that belongs to an object (like .upper())
  • len(): A function that returns the length of a string
  • f-string: A formatted string that lets you insert variables
