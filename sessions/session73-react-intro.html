üöÄ Activity 1: What is React?

Step 1: React is a JavaScript library for building user interfaces. It makes it easier to create interactive webpages!
Step 2: React uses components - reusable pieces of code that represent parts of your webpage.
React is used by many big companies like Facebook, Netflix, and Instagram. It's very popular for building modern websites!

üíª Activity 2: Setting Up React

Step 1: Install Node.js from nodejs.org (if not already installed)
Step 2: Create a new React app using Create React App: 
# In terminal/command prompt:
npx create-react-app my-first-app
cd my-first-app
npm start
Step 3: This will open your app in the browser at http://localhost:3000

üé® Activity 3: Your First Component

Step 1: Open src/App.js and see your first component: 
function App() {
  return (
    <div className="App">
      <h1>Hello, React!</h1>
    </div>
  );
}

export default App;
Step 2: Notice JSX - it looks like HTML but it's JavaScript! JSX lets you write HTML-like code in JavaScript.

üìã Complete Example Program

Example: Here's a complete first React component: 
// App.js
import './App.css';

function App() {
  return (
    <div className="App">
      <header className="App-header">
        <h1>Welcome to React!</h1>
        <p>This is my first React application.</p>
        <div className="info-box">
          <h2>What is React?</h2>
          <ul>
            <li>A JavaScript library</li>
            <li>For building user interfaces</li>
            <li>Uses components</li>
            <li>Very popular!</li>
          </ul>
        </div>
      </header>
    </div>
  );
}

export default App;
Example CSS: Style your component: 
/* App.css */
.App {
  text-align: center;
}

.App-header {
  background-color: #282c34;
  padding: 20px;
  color: white;
  min-height: 100vh;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
}

.info-box {
  background-color: #61dafb;
  color: #282c34;
  padding: 20px;
  border-radius: 10px;
  margin-top: 20px;
  max-width: 500px;
}

.info-box ul {
  text-align: left;
}

üí° Key Concepts

  • React: A JavaScript library for building UIs
  • Component: A reusable piece of UI code
  • JSX: JavaScript syntax that looks like HTML
  • Create React App: Tool to set up React projects
  • npm: Node Package Manager - installs packages
  • className: JSX uses className instead of class
  • export default: Makes component available to other files
