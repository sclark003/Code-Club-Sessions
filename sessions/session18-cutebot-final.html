üé® Project Ideas

Here are some ideas to get you started. Feel free to come up with your own or modify these!

Option 1: Smart Delivery Robot

  • Follows a line to a destination
  • Stops when it detects an obstacle
  • Plays a sound to announce arrival
  • Shows a message on the micro:bit screen
  • Returns to start when button is pressed

Option 2: Obstacle Course Navigator

  • Starts when button A is pressed
  • Moves forward while avoiding obstacles
  • Follows a line when available
  • Shows distance to obstacles on screen
  • Plays celebration music when it reaches the end

Option 3: Light-Seeking Robot

  • Uses light sensor to find the brightest area
  • Moves toward light
  • Avoids obstacles while seeking light
  • Stops and celebrates when it finds bright light
  • Shows light level on micro:bit screen

Option 4: Remote Control with Safety

  • Controlled by micro:bit buttons and tilt
  • Has obstacle detection safety feature
  • Automatically stops if obstacle is too close
  • Shows status on screen (moving, stopped, obstacle detected)
  • Speed control with buttons
  • Emergency stop when shaken

Option 5: Line Follower with Obstacle Avoidance

  • Follows a line automatically
  • Detects obstacles ahead
  • Stops, backs up, and goes around obstacles
  • Returns to line following after avoiding obstacle
  • Shows what it's doing on the screen

Option 6: Your Own Idea!

Think about what you'd like your robot to do. What problem can it solve? What would be fun or useful?

üìù Planning Your Project

Step 1: Choose your project idea
Step 2: List the features you want:
  • What sensors will you use? (line following, ultrasonic, light, etc.)
  • How will you control it? (buttons, automatic, remote control)
  • What will it display? (distance, status, messages)
  • What sounds will it make? (warnings, celebrations, music)
  • What variables do you need?
Step 3: Draw or write down your plan
Step 4: Start building one feature at a time
Step 5: Test each feature as you add it
Step 6: Combine features and test everything together

üîß Tips for Success

Start simple! Get one feature working perfectly, then add more. Don't try to build everything at once.
Test often! After each new feature, download and test on your Cutebot to make sure it works.
Use comments! In MakeCode, you can add comment blocks to explain what your code does.
Debug step by step! If something doesn't work, test each part separately to find the problem.
Be creative! Add your own ideas and make it unique to you!

üéØ Challenge Levels
