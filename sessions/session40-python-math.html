üöÄ Activity 1: Basic Arithmetic Operators

Step 1: Python can do all the basic math operations. Try these in IDLE: 
# Addition
print(5 + 3)

# Subtraction
print(10 - 4)

# Multiplication
print(6 * 7)

# Division
print(15 / 3)

# Integer division (rounds down)
print(17 // 5)

# Modulo (remainder)
print(17 % 5)

# Exponentiation (power)
print(2 ** 3)  # 2 to the power of 3
Step 2: Run each line and see what the result is
The // operator does "floor division" - it divides and rounds down to the nearest whole number. The % operator gives you the remainder after division!

üíª Activity 2: Using Variables in Math

Step 1: You can use variables in calculations: 
num1 = 10
num2 = 3

addition = num1 + num2
subtraction = num1 - num2
multiplication = num1 * num2
division = num1 / num2

print(f"{num1} + {num2} = {addition}")
print(f"{num1} - {num2} = {subtraction}")
print(f"{num1} * {num2} = {multiplication}")
print(f"{num1} / {num2} = {division}")
Step 2: Try changing the values of num1 and num2 and see what happens

üé® Activity 3: Order of Operations

Step 1: Python follows the same order of operations as maths (PEMDAS/BODMAS): 
# Multiplication happens before addition
result1 = 2 + 3 * 4
print(result1)  # 14, not 20!

# Use parentheses to change the order
result2 = (2 + 3) * 4
print(result2)  # 20
Step 2: Try these examples: 
print(10 - 2 * 3)
print((10 - 2) * 3)
print(2 ** 3 + 1)
print(2 ** (3 + 1))

üìä Activity 4: The Math Module

Step 1: Python has a math module with advanced functions. First, you need to import it: 
import math

# Square root
print(math.sqrt(16))

# Power (same as **)
print(math.pow(2, 3))

# Pi (3.14159...)
print(math.pi)

# Round up
print(math.ceil(4.3))  # 5

# Round down
print(math.floor(4.7))  # 4
Step 2: Try calculating the area of a circle: 
import math

radius = 5
area = math.pi * radius ** 2
print(f"The area of a circle with radius {radius} is {area}")

üéØ Activity 5: Building a Calculator

Step 1: Create a simple calculator program: 
print("Simple Calculator")
print("=" * 20)

num1 = float(input("Enter first number: "))
operator = input("Enter operator (+, -, *, /): ")
num2 = float(input("Enter second number: "))

if operator == "+":
    result = num1 + num2
elif operator == "-":
    result = num1 - num2
elif operator == "*":
    result = num1 * num2
elif operator == "/":
    result = num1 / num2
else:
    result = "Invalid operator"

print(f"{num1} {operator} {num2} = {result}")
Step 2: Run the program and test it with different numbers and operators
Create a program that:
  • Asks for the radius of a circle
  • Calculates and displays the area
  • Calculates and displays the circumference
  • Uses the math module for pi

üí° Key Concepts

  • Arithmetic Operators: + (add), - (subtract), * (multiply), / (divide)
  • Integer Division (//): Divides and rounds down to whole number
  • Modulo (%): Returns the remainder after division
  • Exponentiation (**): Raises a number to a power
  • Operator Precedence: The order in which operations are performed
  • Module: A collection of functions you can import and use
  • import: A keyword that brings in code from modules
