ğŸ’» Activity 2: Creating a Simple Line Chart

Step 1: Create your first chart: 
import matplotlib.pyplot as plt

# Data
x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 8, 10]

# Create the chart
plt.plot(x, y)
plt.title("Simple Line Chart")
plt.xlabel("X values")
plt.ylabel("Y values")
plt.show()
Step 2: This will open a window showing your chart!

ğŸ¨ Activity 3: Bar Chart

Step 1: Create a bar chart: 
import matplotlib.pyplot as plt

# Data
subjects = ["Maths", "Science", "English", "History"]
scores = [85, 90, 88, 82]

# Create bar chart
plt.bar(subjects, scores, color=['red', 'blue', 'green', 'orange'])
plt.title("Test Scores by Subject")
plt.xlabel("Subjects")
plt.ylabel("Scores")
plt.ylim(0, 100)  # Set y-axis range
plt.show()

ğŸ“Š Activity 4: Pie Chart

Step 1: Create a pie chart: 
import matplotlib.pyplot as plt

# Data
fruits = ["Apple", "Banana", "Orange", "Grape"]
quantities = [30, 25, 20, 25]
colours = ['red', 'yellow', 'orange', 'purple']

# Create pie chart
plt.pie(quantities, labels=fruits, colors=colours, autopct='%1.1f%%')
plt.title("Fruit Distribution")
plt.show()

ğŸ¯ Activity 5: Multiple Lines on One Chart

Step 1: Compare multiple datasets: 
import matplotlib.pyplot as plt

# Data for two students
months = ["Jan", "Feb", "Mar", "Apr", "May"]
sarah_scores = [85, 87, 90, 88, 92]
alex_scores = [82, 85, 88, 90, 91]

# Plot both lines
plt.plot(months, sarah_scores, marker='o', label='Sarah')
plt.plot(months, alex_scores, marker='s', label='Alex')
plt.title("Test Scores Over Time")
plt.xlabel("Month")
plt.ylabel("Score")
plt.legend()  # Show legend
plt.grid(True)  # Add grid
plt.show()

ğŸŒŸ Activity 6: Saving Charts

Step 1: Save your chart as an image: 
import matplotlib.pyplot as plt

# Create a chart
x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 8, 10]
plt.plot(x, y)
plt.title("My Chart")
plt.xlabel("X")
plt.ylabel("Y")

# Save as image

print("Chart saved as my_chart.png")

ğŸ’¡ Key Concepts

  • matplotlib: Python library for creating charts and graphs
  • plt.plot(): Creates a line chart
  • plt.bar(): Creates a bar chart
  • plt.pie(): Creates a pie chart
  • plt.title(): Adds a title to the chart
  • plt.xlabel()/ylabel(): Adds axis labels
  • plt.legend(): Shows a legend for multiple datasets
  • plt.show(): Displays the chart
  • plt.savefig(): Saves the chart as an image file
