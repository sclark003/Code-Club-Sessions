üöÄ Activity 1: What is Web Development?

Step 1: Web development is creating websites and web applications that people can view in their browsers.
Step 2: There are three main languages:
  • HTML - The structure (like the skeleton of a house)
  • CSS - The styling (like paint and decoration)
  • JavaScript - The behaviour (like making things interactive)
Step 3: Together, they create beautiful, interactive websites!

üíª Activity 2: Setting Up Your Code Editor

Step 1: You need a code editor to write your code. Here are some free options:
  • Visual Studio Code - Very popular and free (code.visualstudio.com)
  • Notepad++ - Simple and lightweight (Windows)
  • Atom - Free and customisable
Step 2: For this course, we recommend Visual Studio Code - it's free and has lots of helpful features!
You can also use any text editor, but code editors have features like syntax highlighting that make coding easier!

üé® Activity 3: Your First Webpage

Step 1: Open your code editor and create a new file called index.html
Step 2: Type this code: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first webpage!</p>
</body>
</html>
Step 3: Save the file (make sure it ends with .html)
Step 4: Double-click the file to open it in your web browser - you should see your webpage!

üìù Activity 4: Understanding HTML Structure

Step 1: Let's break down what each part means:
  • <!DOCTYPE html> - Tells the browser this is an HTML document
  • <html> - The root element that contains everything
  • <head> - Contains information about the page (not visible)
  • <title> - The text that appears in the browser tab
  • <body> - Contains the visible content
  • <h1> - A heading (largest text)
  • <p> - A paragraph (regular text)

üìã Complete Example Program

Example: Here's a complete first webpage: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Welcome to My First Webpage!</h1>
    <p>This is my first attempt at creating a website.</p>
    <p>I'm learning HTML, CSS, and JavaScript!</p>
    
    <h2>About Me</h2>
    <p>My name is Sarah and I'm learning web development!</p>
    
    <h2>What I'm Learning</h2>
    <ul>
        <li>HTML - Structure</li>
        <li>CSS - Styling</li>
        <li>JavaScript - Interactivity</li>
    </ul>
</body>
</html>

üí° Key Concepts

  • Web Development: Creating websites and web applications
  • HTML: HyperText Markup Language - defines the structure
  • CSS: Cascading Style Sheets - defines the appearance
  • JavaScript: A programming language - adds interactivity
  • Code Editor: A program for writing code
  • Browser: A program that displays webpages (Chrome, Firefox, etc.)
  • Element: A piece of HTML code (like <h1> or <p>)
  • Tag: The HTML code that creates an element
