üöÄ Activity 1: What is JavaScript?

Step 1: JavaScript makes webpages interactive - it can respond to clicks, change content, validate forms, and much more!
Step 2: HTML = Structure, CSS = Style, JavaScript = Behaviour!

üíª Activity 2: Adding JavaScript

Step 1: Add JavaScript in a <script> tag: 
<script>
    console.log("Hello, JavaScript!");
</script>
Step 2: Open your browser's Developer Tools (F12) and check the Console tab to see the message!

üé® Activity 3: Your First Interactive Element

Step 1: Create a button that shows an alert: 
<button onclick="alert('Hello!')">Click Me!</button>
Step 2: Or use a script: 
<button id="myButton">Click Me!</button>
<script>
    document.getElementById('myButton').onclick = function() {
        alert('Button clicked!');
    };
</script>

üìã Complete Example Program

Example: Here's a complete interactive webpage: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First JavaScript Page</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            text-align: center;
            padding: 50px;
        }
        button {
            padding: 15px 30px;
            font-size: 18px;
            background-color: #0066cc;
            color: white;
            border: none;
            border-radius: 5px;
            cursor: pointer;
            margin: 10px;
        }
        button:hover {
            background-color: #0052a3;
        }
        #message {
            margin-top: 20px;
            font-size: 24px;
            color: #009900;
        }
    </style>
</head>
<body>
    <h1>JavaScript Interactive Page</h1>
    <button onclick="showAlert()">Show Alert</button>
    <button onclick="changeText()">Change Text</button>
    <button onclick="showTime()">Show Time</button>
    
    <p id="message">Click a button to see JavaScript in action!</p>
    
    <script>
        function showAlert() {
            alert('Hello from JavaScript!');
        }
        
        function changeText() {
            document.getElementById('message').textContent = 'Text changed with JavaScript!';
        }
        
        function showTime() {
            const now = new Date();
            document.getElementById('message').textContent = 'Current time: ' + now.toLocaleTimeString();
        }
        
        // Log to console when page loads
        console.log('Page loaded successfully!');
    </script>
</body>
</html>
Create an interactive page with:
  • A button that changes the background colour
  • A button that displays a random number
  • A button that shows/hides an element

üí° Key Concepts

  • JavaScript: A programming language that adds interactivity
  • Script tag: <script> - where JavaScript code goes
  • console.log(): Displays messages in the browser console
  • alert(): Shows a pop-up message
  • onclick: An event that happens when something is clicked
  • function: A block of code that performs a task
  • getElementById(): Finds an element by its ID
