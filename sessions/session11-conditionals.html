ü§î What is Conditional Logic?

Conditional logic lets your program make decisions! It's like asking "IF this is true, THEN do this, ELSE do that." This is one of the most important concepts in programming!

Step 2: Create a variable called "age" and set it to 10 in "on start"
Step 3: Add "on button A pressed"
Step 4: Add an "if" block from the Logic category
Step 5: Drag the "age" variable into the if block, and use the comparison operator (>) to check if age > 12
Step 6: Inside the if block, add "show string" that says "Teenager!"
Step 7: Test it - change the age value and see what happens!

üí° Key Concepts

  • If Statement: Checks a condition and runs code only if it's true
  • Else Statement: Runs code when the if condition is false
  • Condition: A true/false question (like "is age > 12?")
  • Comparison Operator: Symbols that compare values (>, <, =, ‚â†)
  • Boolean: A value that is either true or false
  • Nested Conditions: Putting if statements inside other if statements
