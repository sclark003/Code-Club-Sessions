ğŸš€ Activity 1: Looping Through Lists

Step 1: A for loop repeats code for each item in a collection: 
fruits = ["apple", "banana", "orange"]

for fruit in fruits:
    print(fruit)
    print("I like " + fruit)
Step 2: The variable fruit takes each value from the list, one at a time
The variable name after "for" can be anything you want - it's just a name for each item as you loop through!

ğŸ’» Activity 2: Looping Through Strings

Step 1: You can loop through strings character by character: 
word = "Python"

for letter in word:
    print(letter)
Step 2: Try counting letters: 
word = "Hello"
count = 0

for letter in word:
    if letter == "l":
        count = count + 1

print(f"The letter 'l' appears {count} times.")

ğŸ¨ Activity 3: The range() Function

Step 1: Use range() to loop a specific number of times: 
# Loop 5 times (0 to 4)
for i in range(5):
    print(i)

# Loop from 1 to 10
for i in range(1, 11):
    print(i)

# Loop with steps
for i in range(0, 10, 2):  # 0, 2, 4, 6, 8
    print(i)
Step 2: Create a countdown: 
for i in range(10, 0, -1):
    print(i)
print("Blast off!")

ğŸ“Š Activity 4: Nested Loops

Step 1: You can put loops inside other loops: 
# Multiplication table
for i in range(1, 6):
    for j in range(1, 6):
        result = i * j
        print(f"{i} x {j} = {result}")
    print()  # Blank line between tables

ğŸ¯ Activity 5: Using enumerate()

Step 1: Sometimes you need both the item and its index: 
fruits = ["apple", "banana", "orange"]

for index, fruit in enumerate(fruits):
    print(f"{index + 1}. {fruit}")

for number in numbers:
    if number > largest:
        largest = number

ğŸ’¡ Key Concepts

  • for loop: Repeats code for each item in a collection
  • range(): Generates a sequence of numbers
  • Iteration: One time through the loop
  • Loop variable: The variable that holds each item during the loop
  • Nested loop: A loop inside another loop
  • enumerate(): Gets both index and value when looping
  • Indentation: Code inside the loop must be indented
