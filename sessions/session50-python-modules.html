üöÄ Activity 1: Importing Modules

Step 1: A module is a file containing Python code. Import it with import: 
import math

print(math.pi)
print(math.sqrt(16))
print(math.pow(2, 3))
Step 2: You can import specific functions: 
from math import sqrt, pi

print(pi)
print(sqrt(25))
Python comes with many built-in modules - you don't need to install them!

üíª Activity 2: The random Module

Step 1: The random module generates random numbers: 
import random

# Random integer between 1 and 10
number = random.randint(1, 10)
print(number)

# Random float between 0 and 1
decimal = random.random()
print(decimal)

# Choose a random item from a list
fruits = ["apple", "banana", "orange"]
choice = random.choice(fruits)
print(choice)

# Shuffle a list
numbers = [1, 2, 3, 4, 5]
random.shuffle(numbers)
print(numbers)

üé® Activity 3: The datetime Module

Step 1: Work with dates and times: 
from datetime import datetime, date, timedelta

# Current date and time
now = datetime.now()
print(f"Current time: {now}")

# Just the date
today = date.today()
print(f"Today: {today}")

# Format the date
formatted = now.strftime("%Y-%m-%d %H:%M:%S")
print(f"Formatted: {formatted}")

# Add days
future = today + timedelta(days=7)
print(f"One week from today: {future}")

üìä Activity 4: The os Module

Step 1: Interact with the operating system: 
import os

# Get current directory
current_dir = os.getcwd()
print(f"Current directory: {current_dir}")

# List files in directory
files = os.listdir(".")
print("Files in current directory:")
for file in files:
    print(f"  - {file}")

# Check if file exists
if os.path.exists("my_file.txt"):
    print("File exists!")
else:
    print("File doesn't exist.")

üåü Activity 6: Creating Your Own Module

Step 1: You can create your own modules! Create a file called my_module.py: 
# my_module.py
def greet(name):
    return f"Hello, {name}!"

def add(a, b):
    return a + b

PI = 3.14159
Step 2: Then use it in another file: 
import my_module

print(my_module.greet("Sarah"))
print(my_module.add(5, 3))
print(my_module.PI)
Create a program that:
  • Uses random to generate a random password
  • Uses datetime to display the current date and time
  • Uses os to check if a file exists
  • Creates your own module with useful functions

üí° Key Concepts

  • Module: A file containing Python code that can be imported
  • Library: A collection of modules
  • import: Keyword to bring in a module
  • Built-in modules: Modules that come with Python (math, random, datetime, etc.)
  • External libraries: Modules you need to install (using pip)
  • pip: Package installer for Python
  • from...import: Import specific functions from a module
