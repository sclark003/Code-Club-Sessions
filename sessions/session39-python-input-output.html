üöÄ Activity 1: Getting Text Input

Step 1: The input() function lets your program ask the user for information
Step 2: Create a new Python file and type this: 
name = input("What is your name? ")
print("Hello, " + name + "!")
Step 3: Run the program. It will wait for you to type something and press Enter
Step 4: Type your name and press Enter. The program will greet you!
The text inside the input() function is called a "prompt" - it tells the user what to type!

üíª Activity 2: Multiple Inputs

Step 1: You can ask for multiple pieces of information: 
first_name = input("What is your first name? ")
last_name = input("What is your last name? ")
print(f"Nice to meet you, {first_name} {last_name}!")
Step 2: Run the program and answer both questions
Step 3: Try creating a program that asks for:
  • Your favourite colour
  • Your favourite food
  • Your favourite hobby
Then print them all in a sentence!

üé® Activity 3: Converting Input Types

Step 1: By default, input() always returns a string (text). If you want a number, you need to convert it: 
age = input("How old are you? ")
print(type(age))  # This will show it's a string

age = int(input("How old are you? "))
print(type(age))  # Now it's an integer!
Step 2: Try this program: 
age = int(input("How old are you? "))
next_year = age + 1
print(f"Next year you will be {next_year} years old!")
Step 3: For decimal numbers, use float(): 
height = float(input("What is your height in metres? "))
print(f"You are {height} metres tall!")

Step 2: Run the program and fill in the story with your answers
Step 3: Try adding more questions to make the story longer!

üìã Complete Example Program

Example: Here's a complete interactive program: 
# interactive_program.py
# An interactive program that gets user input

print("=" * 40)
print("Welcome to the Interactive Program!")
print("=" * 40)

# Get user's name
name = input("What is your name? ")
print(f"Hello, {name}! Nice to meet you.")

# Get user's age
age = int(input("How old are you? "))
print(f"You are {age} years old.")

# Get user's favourite colour
favourite_colour = input("What is your favourite colour? ")
print(f"{favourite_colour} is a great colour!")

# Get user's favourite number
favourite_number = int(input("What is your favourite number? "))
double = favourite_number * 2
print(f"Double your favourite number is {double}!")

# Create a summary
print("\n" + "=" * 40)
print("Summary")
print("=" * 40)
print(f"Name: {name}")
print(f"Age: {age}")
print(f"Favourite colour: {favourite_colour}")
print(f"Favourite number: {favourite_number}")
print(f"Double of favourite number: {double}")

print("\nThank you for using the program!")

Create a program that:
  • Asks for the user's name
  • Asks for their birth year
  • Calculates and displays their age
  • Asks how many pets they have
  • Creates a personalised message with all this information

Example solution:

# Age calculator program
from datetime import datetime

print("Personal Information Program")
print("=" * 35)

name = input("What is your name? ")
birth_year = int(input("What year were you born? "))
current_year = datetime.now().year
age = current_year - birth_year

pets = int(input("How many pets do you have? "))

print("\n" + "=" * 35)
print(f"Hello, {name}!")
print(f"You were born in {birth_year}")
print(f"You are {age} years old")
print(f"You have {pets} pet(s)")

if pets > 0:
    print(f"That's great! {pets} pet(s) must be fun!")
else:
    print("Maybe you'll get a pet one day!")

üí° Key Concepts

  • input(): A function that gets text from the user
  • Prompt: The message shown to the user asking for input
  • Type Conversion: Changing data from one type to another (e.g., string to integer)
  • int(): Converts a value to an integer (whole number)
  • float(): Converts a value to a float (decimal number)
  • Interactive Program: A program that responds to user input
