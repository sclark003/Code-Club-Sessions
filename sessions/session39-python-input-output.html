üöÄ Activity 1: Getting Text Input

Step 1: The input() function lets your program ask the user for information
Step 2: Create a new Python file and type this: 
name = input("What is your name? ")
print("Hello, " + name + "!")
Step 3: Run the program. It will wait for you to type something and press Enter
Step 4: Type your name and press Enter. The program will greet you!
The text inside the input() function is called a "prompt" - it tells the user what to type!

üíª Activity 2: Multiple Inputs

Step 1: You can ask for multiple pieces of information: 
first_name = input("What is your first name? ")

print(f"Nice to meet you, {first_name} {last_name}!")
Step 2: Run the program and answer both questions
Step 3: Try creating a program that asks for:
  • Your favourite colour
  • Your favourite food
  • Your favourite hobby
Then print them all in a sentence!

üé® Activity 3: Converting Input Types

Step 1: By default, input() always returns a string (text). If you want a number, you need to convert it: 
age = input("How old are you? ")
print(type(age))  # This will show it's a string

age = int(input("How old are you? "))
print(type(age))  # Now it's an integer!
Step 2: Try this program: 
age = int(input("How old are you? "))
next_year = age + 1
print(f"Next year you will be {next_year} years old!")
Step 3: For decimal numbers, use float(): 
height = float(input("What is your height in metres? "))
print(f"You are {height} metres tall!")

üìä Activity 4: Creating an Interactive Story

Step 1: Let's create a simple interactive story: 
print("Welcome to the Adventure Story!")
name = input("What is your name? ")
place = input("Where would you like to go? ")
animal = input("What is your favourite animal? ")

print(f"\nOnce upon a time, {name} went to {place}.")
print(f"While there, {name} met a friendly {animal}.")
print(f"{name} and the {animal} became best friends!")
print("The end!")
Step 2: Run the program and fill in the story with your answers
Step 3: Try adding more questions to make the story longer!

Step 2: Try modifying it to do subtraction, multiplication, or division

üí° Key Concepts

  • input(): A function that gets text from the user
  • Prompt: The message shown to the user asking for input
  • Type Conversion: Changing data from one type to another (e.g., string to integer)
  • int(): Converts a value to an integer (whole number)
  • float(): Converts a value to a float (decimal number)
  • Interactive Program: A program that responds to user input
