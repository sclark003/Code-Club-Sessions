ğŸ“¡ What is Radio Communication?

The micro:bit has a built-in radio that can send and receive messages wirelessly! This lets micro:bits "talk" to each other without wires. You can send numbers, text, or other data between micro:bits.

For radio to work, all micro:bits need to be on the same "radio group" (like a channel). Make sure everyone uses the same group number!

ğŸš€ Activity 1: Set Up Radio

Step 1: Open MakeCode and create a new project called "Radio"
Step 2: Go to the "Radio" category (you might need to add it - click the gear icon)
Step 3: Find "radio set group" and drag it into "on start"
Step 4: Set the group number to 1 (or any number - everyone needs the same number!)
Radio groups are like walkie-talkie channels. Everyone on the same group can hear each other. Use different groups to avoid interference!

ğŸ¨ Activity 2: Send a Message

Step 1: Add "on button A pressed"
Step 2: Add "radio send string" from the Radio category
Step 3: Type a message like "Hello!" in the string block
Step 4: Add "show string" that says "Sent!" so you know it worked

ğŸ¯ Activity 3: Receive a Message

Step 1: Add "on radio received receivedString" block
Step 2: Inside it, add "show string"
Step 3: Drag the "receivedString" variable into the "show string" block
Step 4: Now when someone sends a message, your micro:bit will display it!
Step 5: Test with a partner - one person presses button A to send, the other receives the message!

ğŸŒŸ Activity 4: Send Numbers

Step 1: Create a variable called "score"
Step 2: Add "on button A pressed" that increases the score
Step 3: Add "on button B pressed" that sends the score using "radio send number"
Step 4: Add "on radio received receivedNumber" to receive and display the number
Step 5: Now you can share scores between micro:bits!

ğŸ® Activity 5: Simple Chat

Step 1: Create a program where:
  • Button A sends "Hi!"
  • Button B sends "Bye!"
  • Shaking sends "Help!"
Step 2: All received messages are displayed on screen
Step 3: Test with a partner - you now have a simple chat system!
Create a multiplayer game where:
  • Each player has a score
  • Pressing buttons increases your score
  • Shaking sends your score to everyone
  • Everyone can see each other's scores
  • Hint: Use variables to store your own score and received scores!

ğŸ’¡ Key Concepts

  • Radio: Wireless communication between devices
  • Radio Group: A channel number - devices on the same group can communicate
  • Send: Transmitting a message to other micro:bits
  • Receive: Getting a message from another micro:bit
  • Wireless: Communication without wires or cables
