üöÄ Activity 1: Common Events

Step 1: Common events include:
  • click - When something is clicked
  • input - When input field changes
  • change - When value changes
  • mouseover - When mouse hovers
  • keydown - When key is pressed

üíª Activity 2: Adding Event Listeners

Step 1: Add event listeners: 
// Method 1: Inline
<button onclick="doSomething()">Click Me</button>

// Method 2: addEventListener (better!)
button.addEventListener('click', function() {
    console.log('Clicked!');
});

üìã Complete Example Program

Example: Here's a complete interactive example: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>JavaScript Events</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            max-width: 600px;
            margin: 50px auto;
            padding: 20px;
        }
        #box {
            width: 200px;
            height: 200px;
            background-color: #0066cc;
            margin: 20px auto;
            border-radius: 10px;
            transition: all 0.3s;
        }
        button, input {
            padding: 10px;
            margin: 5px;
            font-size: 16px;
        }
        button {
            background-color: #0066cc;
            color: white;
            border: none;
            border-radius: 5px;
            cursor: pointer;
        }
    </style>
</head>
<body>
    <h1>Event Examples</h1>
    
    <button id="clickBtn">Click Me!</button>
    <button id="hoverBtn">Hover Over Me!</button>
    <br>
    <input type="text" id="textInput" placeholder="Type something...">
    <br>
    <div id="box"></div>
    <div id="output"></div>
    
    <script>
        // Click event
        document.getElementById('clickBtn').addEventListener('click', function() {
            document.getElementById('output').textContent = 'Button was clicked!';
            document.getElementById('box').style.backgroundColor = 'green';
        });
        
        // Mouseover event
        document.getElementById('hoverBtn').addEventListener('mouseover', function() {
            this.style.backgroundColor = 'orange';
        });
        
        document.getElementById('hoverBtn').addEventListener('mouseout', function() {
            this.style.backgroundColor = '#0066cc';
        });
        
        // Input event
        document.getElementById('textInput').addEventListener('input', function(e) {
            document.getElementById('output').textContent = 'You typed: ' + e.target.value;
        });
        
        // Box click
        document.getElementById('box').addEventListener('click', function() {
            const colours = ['red', 'blue', 'green', 'purple', 'orange'];
            const randomColour = colours[Math.floor(Math.random() * colours.length)];
            this.style.backgroundColor = randomColour;
        });
    </script>
</body>
</html>
Create an interactive page with:
  • A button that changes colour on hover
  • An input that shows character count as you type
  • A div that changes size when clicked

üí° Key Concepts

  • Event: An action that happens (click, type, hover)
  • Event Listener: Code that waits for an event
  • addEventListener(): Method to add event listeners
  • Event Object: Contains information about the event
  • this: Refers to the element that triggered the event
