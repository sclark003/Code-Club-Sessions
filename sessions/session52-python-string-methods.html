üöÄ Activity 1: Splitting and Joining

Step 1: Split strings into lists and join lists into strings: 
# Split a string
sentence = "Hello, how are you?"
words = sentence.split()
print(words)  # ['Hello,', 'how', 'are', 'you?']

# Split by specific character
data = "apple,banana,orange"
fruits = data.split(",")
print(fruits)  # ['apple', 'banana', 'orange']

# Join a list into a string
words = ["Hello", "world", "Python"]
sentence = " ".join(words)
print(sentence)  # "Hello world Python"

# Join with different separator
email_parts = ["user", "example", "com"]
email = ".".join(email_parts)
print(email)  # "user.example.com"

üíª Activity 2: String Validation

Step 1: Check if strings meet certain conditions: 
text = "Hello123"

# Check if all characters are alphabetic
print(text.isalpha())  # False (has numbers)

# Check if all characters are digits
print("12345".isdigit())  # True

# Check if all characters are alphanumeric
print(text.isalnum())  # True

# Check if all characters are lowercase
print("hello".islower())  # True
print("Hello".islower())  # False

# Check if all characters are uppercase
print("HELLO".isupper())  # True

# Check if string starts/ends with something
print("Hello".startswith("He"))  # True
print("Hello".endswith("lo"))  # True

üé® Activity 3: String Cleaning

Step 1: Clean and format strings: 
text = "  Hello, World!  "

# Remove whitespace from ends
cleaned = text.strip()
print(f"'{cleaned}'")

# Remove from left only
left_cleaned = text.lstrip()
print(f"'{left_cleaned}'")

# Remove from right only
right_cleaned = text.rstrip()
print(f"'{right_cleaned}'")

# Replace characters
new_text = text.replace("World", "Python")
print(new_text)

# Remove all spaces
no_spaces = text.replace(" ", "")
print(no_spaces)

üìä Activity 4: Case Conversion

Step 1: Change the case of strings: 
text = "Hello, World!"

# Convert to uppercase
print(text.upper())  # "HELLO, WORLD!"

# Convert to lowercase
print(text.lower())  # "hello, world!"

# Title case (first letter of each word)
print(text.title())  # "Hello, World!"

# Capitalise (first letter only)
print(text.capitalize())  # "Hello, world!"

# Swap case
print(text.swapcase())  # "hELLO, wORLD!"

üéØ Activity 5: Finding and Counting

Step 1: Find and count occurrences: 
text = "Hello, hello, hello!"

# Find first occurrence
position = text.find("hello")
print(position)  # 7

# Find from specific position
position = text.find("hello", 10)
print(position)

# Count occurrences
count = text.count("hello")
print(count)  # 3

# Check if substring exists
if "hello" in text:
    print("Found!")

# Find index (raises error if not found)
try:
    index = text.index("world")
except ValueError:
    print("Not found!")

üåü Activity 6: Practical Example

Step 1: Create a text processing program: 
def process_text(text):
    # Remove extra spaces
    text = " ".join(text.split())
    
    # Convert to title case
    text = text.title()
    
    # Count words
    words = text.split()
    word_count = len(words)
    
    # Count characters
    char_count = len(text)
    
    return text, word_count, char_count

user_text = input("Enter some text: ")
processed, words, chars = process_text(user_text)

print(f"\nProcessed text: {processed}")
print(f"Word count: {words}")
print(f"Character count: {chars}")

üí° Key Concepts

  • split(): Divides a string into a list
  • join(): Combines a list into a string
  • strip(): Removes whitespace from ends
  • replace(): Replaces characters in a string
  • isalpha(), isdigit(), etc.: Check string properties
  • find(): Finds position of substring
  • count(): Counts occurrences of substring
  • Case methods: upper(), lower(), title(), capitalize()
