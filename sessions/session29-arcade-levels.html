üåç Game Worlds

Today we'll learn to create beautiful backgrounds and multiple levels for your games! This makes games more interesting and challenging.

üöÄ Activity 1: Set a Background

Step 1: Open MakeCode Arcade and create a new project called "Levels"
Step 2: Look in the "Scene" category
Step 3: Find "set background image to"
Step 4: Add it in "on start"
Step 5: Click on the image to choose a background (forest, space, city, etc.)
Step 6: You can also draw your own custom background!
Backgrounds set the scene for your game. Choose one that matches your game's theme!

üé® Activity 2: Background Colour

Step 1: Instead of an image, you can use a solid colour
Step 2: Use "set background color to" (from Scene category)
Step 3: Choose a colour that matches your game
Step 4: Try different colours for different levels!

üéØ Activity 3: Create Multiple Levels

Step 1: Create a variable called "level" (from Variables category)
Step 2: Set level to 1 in "on start"
Step 3: Create a function called "startLevel" (from Functions category)
Step 4: Inside the function, use an if statement to set different backgrounds based on level:
  • If level = 1: Set background to forest
  • If level = 2: Set background to space
  • If level = 3: Set background to city
Step 5: Call this function in "on start" and when level changes

üåü Activity 4: Level Progression

Step 1: When player reaches a certain score, advance to next level
Step 2: Check if score equals 10 (or your target)
Step 3: If true:
  • Change level by 1
  • Call "startLevel" function
  • Reset score to 0 (or keep it)
  • Show text "Level Up!"
  • Spawn new enemies or items
Step 4: Make each level harder (more enemies, faster movement, etc.)

üéÆ Activity 5: Display Level

Step 1: Show the current level on screen
Step 2: Use "set level to" (from Info category) to display level
Step 3: Or create a text sprite that shows "Level: " + level variable
Step 4: Update it when level changes

üåÜ Activity 6: Different Level Themes

Step 1: Make each level feel different:
  • Level 1: Forest theme - green background, tree sprites
  • Level 2: Space theme - dark background, star sprites
  • Level 3: City theme - gray background, building sprites
Step 2: Change sprite colours to match the theme
Step 3: Use different music for each level
Step 4: Adjust difficulty (more enemies, faster speed) as levels increase
Create a multi-level game:
  • 3 different levels with unique backgrounds
  • Each level has different enemies and collectibles
  • Level progression when score target is reached
  • Level number displayed on screen
  • Each level gets progressively harder
  • Win condition: Complete all 3 levels
  • Different music/theme for each level

üí° Key Concepts

  • Background: The image or colour behind your game
  • Level: A stage or section of your game
  • Level Progression: Moving from one level to the next
  • Theme: The visual style and setting of a level
  • Difficulty Scaling: Making levels progressively harder
  • Function: Reusable code that you can call multiple times
