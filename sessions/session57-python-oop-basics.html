üöÄ Activity 1: What are Classes and Objects?

Step 1: A class is like a blueprint, and an object is something made from that blueprint: 
# Class definition (the blueprint)
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def bark(self):
        return f"{self.name} says Woof!"
    
    def get_info(self):
        return f"{self.name} is {self.age} years old"

# Create objects (instances)
dog1 = Dog("Buddy", 3)
dog2 = Dog("Max", 5)

print(dog1.bark())
print(dog2.get_info())
Think of a class as a cookie cutter and objects as the cookies! The cookie cutter (class) defines the shape, and you can make many cookies (objects) from it.

üíª Activity 2: Creating a Simple Class

Step 1: Create a Student class: 
class Student:
    def __init__(self, name, age, grade):
        self.name = name
        self.age = age
        self.grade = grade
        self.scores = []
    
    def add_score(self, score):
        self.scores.append(score)
    
    def get_average(self):
        if len(self.scores) == 0:
            return 0
        return sum(self.scores) / len(self.scores)
    
    def get_info(self):
        return f"{self.name}, age {self.age}, grade {self.grade}"

# Create student objects
student1 = Student("Sarah", 12, 7)
student1.add_score(85)
student1.add_score(90)
student1.add_score(88)

print(student1.get_info())
print(f"Average: {student1.get_average():.1f}")

üé® Activity 3: More Class Examples

Step 1: Create a Rectangle class: 
class Rectangle:
    def __init__(self, width, height):
        self.width = width
        self.height = height
    
    def area(self):
        return self.width * self.height
    
    def perimeter(self):
        return 2 * (self.width + self.height)
    
    def is_square(self):
        return self.width == self.height

# Create rectangles
rect1 = Rectangle(5, 5)
rect2 = Rectangle(4, 6)

print(f"Rect1 area: {rect1.area()}")
print(f"Rect1 is square: {rect1.is_square()}")
print(f"Rect2 perimeter: {rect2.perimeter()}")
print(f"Rect2 is square: {rect2.is_square()})

üìä Activity 4: Class Attributes

Step 1: Class attributes are shared by all objects: 
class Car:
    # Class attribute (shared by all cars)
    wheels = 4
    
    def __init__(self, make, model, year):
        # Instance attributes (unique to each car)
        self.make = make
        self.model = model
        self.year = year
        self.speed = 0
    
    def accelerate(self, amount):
        self.speed += amount
    
    def brake(self, amount):
        self.speed = max(0, self.speed - amount)
    
    def get_info(self):
        return f"{self.year} {self.make} {self.model}, speed: {self.speed} km/h"

car1 = Car("Toyota", "Corolla", 2020)
car2 = Car("Honda", "Civic", 2021)

car1.accelerate(50)
car2.accelerate(60)

print(car1.get_info())
print(car2.get_info())
print(f"All cars have {Car.wheels} wheels")

üéØ Activity 5: String Representation

Step 1: Use __str__ to define how objects are printed: 
class Book:
    def __init__(self, title, author, pages):
        self.title = title
        self.author = author
        self.pages = pages
    
    def __str__(self):
        return f"'{self.title}' by {self.author} ({self.pages} pages)"
    
    def is_long(self):
        return self.pages > 300

book1 = Book("Python Guide", "John Smith", 450)
book2 = Book("Short Story", "Jane Doe", 150)

print(book1)  # Uses __str__ method
print(book2)
print(f"Is long: {book1.is_long()}")

üåü Activity 6: Library System

Step 1: Create a simple library system: 
class Book:
    def __init__(self, title, author):
        self.title = title
        self.author = author
        self.is_borrowed = False
    
    def borrow(self):
        if not self.is_borrowed:
            self.is_borrowed = True
            return True
        return False
    
    def return_book(self):
        self.is_borrowed = False

class Library:
    def __init__(self):
        self.books = []
    
    def add_book(self, book):
        self.books.append(book)
    
    def find_book(self, title):
        for book in self.books:
            if book.title == title:
                return book
        return None

# Create library
library = Library()
library.add_book(Book("Python Basics", "Alice"))
library.add_book(Book("Advanced Python", "Bob"))

# Borrow a book
book = library.find_book("Python Basics")
if book:
    if book.borrow():
        print(f"Borrowed: {book.title}")
    else:
        print(f"{book.title} is already borrowed!")

üí° Key Concepts

  • Class: A blueprint for creating objects
  • Object/Instance: A specific example created from a class
  • __init__: Special method that runs when an object is created
  • self: Reference to the current object
  • Method: A function that belongs to a class
  • Attribute: A variable that belongs to an object
  • Class attribute: Shared by all objects of the class
  • Instance attribute: Unique to each object
  • __str__: Defines how an object is converted to a string
