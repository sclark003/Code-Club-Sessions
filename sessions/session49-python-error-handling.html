üöÄ Activity 1: What are Errors?

Step 1: Errors happen when Python can't run your code. Try this: 
# This will cause an error
number = int(input("Enter a number: "))
result = 10 / number
print(result)
Step 2: What happens if you enter 0? Or if you enter text instead of a number?
Errors are normal! The important thing is to handle them gracefully so your program doesn't crash.

üíª Activity 2: Using try/except

Step 1: Use try and except to catch errors: 
try:
    number = int(input("Enter a number: "))
    result = 10 / number
    print(f"10 divided by {number} is {result}")
except:
    print("Something went wrong!")
Step 2: Now the program won't crash - it will show a friendly message instead!

üé® Activity 3: Specific Error Types

Step 1: You can catch specific types of errors: 
try:
    number = int(input("Enter a number: "))
    result = 10 / number
    print(f"Result: {result}")
except ValueError:
    print("That's not a valid number!")
except ZeroDivisionError:
    print("You can't divide by zero!")
except Exception as e:
    print(f"An error occurred: {e}")
Step 2: Each error type gets its own message - much more helpful!

üìä Activity 4: else and finally

Step 1: Use else for code that runs if no error occurs, and finally for code that always runs: 
try:
    number = int(input("Enter a number: "))
    result = 10 / number
except ValueError:
    print("Invalid input!")
except ZeroDivisionError:
    print("Can't divide by zero!")
else:
    print(f"Success! Result is {result}")
finally:
    print("This always runs, whether there was an error or not.")

üéØ Activity 5: File Error Handling

Step 1: Always handle file errors: 
try:
    with open("my_file.txt", "r") as file:
        content = file.read()
        print(content)
except FileNotFoundError:
    print("File not found!")
except PermissionError:
    print("You don't have permission to read this file!")
except Exception as e:
    print(f"An error occurred: {e}")

üåü Activity 6: Robust Input Program

Step 1: Create a program that keeps asking until it gets valid input: 
while True:
    try:
        age = int(input("How old are you? "))
        if age < 0 or age > 120:
            print("Please enter a realistic age!")
            continue
        print(f"You are {age} years old.")
        break
    except ValueError:
        print("Please enter a valid number!")
    except Exception as e:
        print(f"An error occurred: {e}")
Create a calculator program that:
  • Handles division by zero
  • Handles invalid number input
  • Handles invalid operators
  • Keeps asking until the user wants to quit

üí° Key Concepts

  • Error/Exception: A problem that occurs when running code
  • try: Block of code that might cause an error
  • except: Block that handles errors
  • ValueError: Error when value is wrong type
  • ZeroDivisionError: Error when dividing by zero
  • FileNotFoundError: Error when file doesn't exist
  • else: Runs if no error occurred
  • finally: Always runs, regardless of errors
