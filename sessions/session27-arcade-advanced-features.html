‚ö° Power-Up System

Today we'll add exciting features that make your game more interesting! Power-ups give temporary abilities, different enemies create variety, and special effects make everything feel more polished.

üöÄ Activity 1: Create Power-Ups

Step 1: Create a variable "powerUpActive" to track if player has a power-up
Step 2: Create a variable "powerUpType" to track which power-up is active
Step 3: Create power-up sprites (different types):
  • Speed boost (makes player faster)
  • Shield (protects from damage)
  • Double points (score multiplies)
  • Slow motion (enemies move slower)
Step 4: Set their kind to "PowerUp"
Step 5: Make them spawn randomly or at specific locations

üé® Activity 2: Power-Up Effects

Step 1: Create collision handler for Player + PowerUp
Step 2: When collected:
  • Set powerUpActive to true
  • Set powerUpType based on which power-up was collected
  • Destroy the power-up sprite
  • Play sound effect
  • Show visual effect (particles, flash)
Step 3: Apply the power-up effect:
  • Speed boost: Increase player max speed
  • Shield: Set invincibility flag
  • Double points: Set score multiplier to 2
  • Slow motion: Reduce enemy speed
Step 4: Add a timer - power-ups should last 5-10 seconds
Step 5: When timer ends, remove the power-up effect

üéØ Activity 3: Multiple Enemy Types

Step 1: Create different enemy types with different behaviors:
  • Chaser: Follows the player
  • Patroller: Moves back and forth
  • Shooter: Shoots projectiles at player
  • Fast: Moves quickly but predictably
Step 2: Create a function "createEnemy" that takes enemy type as parameter
Step 3: For each enemy type, set different properties:
  • Different sprite image
  • Different speed
  • Different movement pattern
  • Different point value when defeated

üåü Activity 4: Enemy AI Behaviors

Step 1: Create function "updateEnemies" that runs in game loop
Step 2: For Chaser enemies:
  • Calculate direction to player
  • Move toward player
  • Use angle from enemy to player
Step 3: For Patroller enemies:
  • Move in one direction
  • When hitting boundary, reverse direction
  • Or move in a pattern (up-down, left-right)
Step 4: For Shooter enemies:
  • Periodically create projectile sprite
  • Set projectile velocity toward player
  • Projectile damages player on hit

üéÆ Activity 5: Special Abilities

Step 1: Add special ability that player can activate with button
Step 2: Create ability meter (variable that fills up)
Step 3: Ability ideas:
  • Dash: Quick burst of speed in one direction
  • Shield: Temporary invincibility
  • Bomb: Destroy all enemies on screen
  • Slow Time: Slow down all enemies
Step 4: When button pressed and meter is full:
  • Activate ability
  • Reset meter
  • Play sound and visual effect
Step 5: Meter fills up over time or when collecting items

‚ú® Activity 6: Visual Effects

Step 1: Add particle effects:
  • When collecting items: sparkles
  • When hitting enemy: explosion
  • When using ability: special effect
Step 2: Use "start effect" blocks to create visual feedback
Step 3: Add screen shake when big events happen
Step 4: Add color effects (flash screen, change colors)
Create an advanced game with:
  • At least 3 different power-up types with unique effects
  • 4 different enemy types with different AI behaviors
  • Special ability system with meter
  • Visual effects for all major events
  • Power-up timer display on screen
  • Different enemies spawn at different times/levels
  • Boss enemy with unique behavior

üí° Key Concepts

  • Power-Up: Temporary ability that enhances player
  • AI (Artificial Intelligence): Behavior patterns for enemies
  • Enemy Types: Different enemies with different behaviors
  • Special Ability: Player-activated powerful move
  • Particle Effects: Visual effects made of many small sprites
  • Timer System: Tracking time for temporary effects
  • Dynamic Gameplay: Game elements that change during play
