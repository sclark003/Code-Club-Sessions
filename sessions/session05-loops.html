üîÑ What is a Loop?

A loop is a way to repeat actions without writing the same code over and over. Instead of copying and pasting, you tell the computer "do this 5 times" or "keep doing this until something happens".

üöÄ Activity 1: Repeat Loop

Step 1: Open MakeCode and create a new project called "Loops"
Step 2: Go to the "Loops" category and find "repeat 4 times"
Step 3: Drag it into the "on start" block
Step 4: Add a "show icon" block inside the repeat loop
Step 5: Add a "pause" block (from Basic) set to 500ms (half a second)
Step 6: Change the repeat number to 10 and watch the icon flash 10 times!
The pause block gives a delay between actions. Without it, things happen too fast to see!

üé® Activity 2: Counting Loop

Step 1: Find the "for index from 0 to 4" block in the Loops category
Step 2: Drag it into "on start"
Step 3: Inside the loop, add "show number"
Step 4: Drag the "index" variable into the "show number" block
Step 5: Add a pause block. Watch it count from 0 to 4!
Step 6: Change "0 to 4" to "1 to 10" to count higher

üéØ Activity 3: Animation Loop

Step 1: Create a new project called "Animation"
Step 2: Add a "repeat 5 times" loop
Step 3: Inside, add multiple "show icon" blocks with different icons
Step 4: Add "pause 200ms" between each icon
Step 5: The loop will repeat your animation 5 times!

üåü Activity 4: Forever Loop

Step 1: Find the "forever" block in the Loops category
Step 2: This loop runs continuously - it never stops!
Step 3: Add a "show icon" and "pause" inside it
Step 4: Your animation will play forever!
Be careful with forever loops! They run continuously, so make sure to include pause blocks or your micro:bit might freeze!
Create a program that:
  • Uses a loop to show numbers 1, 2, 3, 4, 5
  • Then shows a star icon
  • Repeats this pattern 3 times
  • Hint: You can put loops inside other blocks!

üí° Key Concepts

  • Loop: A way to repeat code without writing it multiple times
  • Repeat Loop: Repeats a specific number of times (like "repeat 5 times")
  • For Loop: Repeats with a counter variable (like "for index from 1 to 10")
  • Forever Loop: Repeats continuously, never stops
  • Index: A variable that counts in a for loop (0, 1, 2, 3...)
