Step 1: Functions can accept input values called parameters:

    print(f"Hello, {name}!")
    print("Welcome to Code Club!")

greet("Sarah")
greet("Alex")
greet("Jordan")
Step 2: Functions can have multiple parameters: 
def introduce(name, age):
    print(f"My name is {name} and I am {age} years old.")

introduce("Sarah", 12)
introduce("Alex", 14)

        return b

print(max_number(20, 3))  # 20

ğŸ’¡ Key Concepts

  • Function: A reusable block of code that performs a specific task
  • def: Keyword used to define a function
  • Parameter: A variable that receives a value when the function is called
  • Argument: The value passed to a function when it's called
  • return: Sends a value back from the function
  • Call: To execute a function by writing its name with parentheses
  • Default parameter: A parameter with a default value if none is provided
