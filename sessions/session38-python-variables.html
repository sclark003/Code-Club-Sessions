ğŸš€ Activity 1: What is a Variable?

Step 1: A variable is like a labelled box where you can store information. You give it a name and put a value inside.
Step 2: Think of it like a name tag - you write your name on it, and whenever someone sees that tag, they know it's you!
Step 3: In Python, you create a variable by giving it a name and using the equals sign (=) to assign a value

ğŸ’» Activity 2: Creating Variables

Step 1: Open Python IDLE and create a new file
Step 2: Type this code: 
name = "Sarah"
age = 12
favourite_colour = "blue"

print(name)
print(age)
print(favourite_colour)
Step 3: Run the program and see what happens! Each variable's value is printed.
Step 4: Change the values to your own information and run it again
Variable names can contain letters, numbers, and underscores, but they can't start with a number. Use descriptive names like student_name instead of sn!

ğŸ¨ Activity 3: Using Variables in Messages

Step 1: You can combine variables with text in print statements: 
name = "Alex"
print("Hello, " + name + "!")
print("Nice to meet you, " + name)
Step 2: Try using f-strings (a special way to format strings): 
name = "Alex"
age = 12
print(f"My name is {name} and I am {age} years old.")
Step 3: Notice the f before the quotes and the curly braces {} around the variable names!

ğŸ“Š Activity 4: Different Data Types

Step 1: Python has different types of data. Let's explore them: 
# String - text (in quotes)
name = "Python"
greeting = "Hello"

# Integer - whole numbers
age = 15
score = 100

# Float - decimal numbers
height = 1.65
temperature = 23.5

# Boolean - True or False
is_student = True
is_weekend = False

print(name, age, height, is_student)
Step 2: Try creating variables of each type with your own values
Step 3: Use the type() function to check what type a variable is: 
name = "Python"
age = 15
print(type(name))
print(type(age))
Create a program that stores information about yourself:
  • Your name (string)

  • Print all this information in a nice message using f-strings

ğŸ’¡ Key Concepts

  • Variable: A named container that stores a value
  • String: Text data, written in quotes (e.g., "Hello")
  • Integer: Whole numbers (e.g., 5, 100, -3)
  • Float: Decimal numbers (e.g., 3.14, 2.5)
  • Boolean: True or False values
  • Assignment: Using = to give a variable a value
  • f-string: A formatted string that lets you insert variables easily
