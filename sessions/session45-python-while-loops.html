üöÄ Activity 1: Basic While Loop

Step 1: A while loop repeats code as long as a condition is True: 
count = 1

while count <= 5:
    print(f"Count: {count}")
    count = count + 1

print("Done!")
Step 2: The loop continues until the condition becomes False
Be careful with while loops! Make sure the condition will eventually become False, or you'll create an infinite loop!

üíª Activity 2: Interactive While Loop

Step 1: While loops are great for interactive programs: 
name = ""

while name == "":
    name = input("What is your name? ")
    if name == "":
        print("Please enter a name!")

print(f"Hello, {name}!")
Step 2: This keeps asking until the user enters something

üé® Activity 3: Using break

Step 1: Use break to exit a loop early: 
while True:
    answer = input("Say 'quit' to exit: ")
    if answer == "quit":
        break
    print(f"You said: {answer}")

print("Goodbye!")
Step 2: while True: creates a loop that runs forever until you break out of it

üìä Activity 4: Using continue

Step 1: Use continue to skip to the next iteration: 
count = 0

while count < 10:
    count = count + 1
    if count % 2 == 0:  # If count is even
        continue  # Skip the rest and go to next iteration
    print(count)  # Only prints odd numbers

üéØ Activity 5: Number Guessing Game

Step 1: Create a number guessing game: 
import random

secret_number = random.randint(1, 100)
guesses = 0

print("I'm thinking of a number between 1 and 100!")

while True:
    guess = int(input("Guess the number: "))
    guesses = guesses + 1
    
    if guess < secret_number:
        print("Too low! Try again.")
    elif guess > secret_number:
        print("Too high! Try again.")
    else:
        print(f"Correct! You guessed it in {guesses} tries!")
        break

üí° Key Concepts

  • while loop: Repeats code as long as a condition is True
  • Condition: The expression that determines if the loop continues
  • Infinite loop: A loop that never stops (usually a mistake!)
  • break: Exits the loop immediately
  • continue: Skips to the next iteration of the loop
  • while True: Creates a loop that runs until you break out
  • Difference from for: while loops repeat based on a condition; for loops repeat for each item in a collection
