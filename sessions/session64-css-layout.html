üöÄ Activity 1: The Box Model

Step 1: Every HTML element is a box with:
  • Content: The actual content (text, images)
  • Padding: Space inside the box (around content)
  • Border: The edge of the box
  • Margin: Space outside the box (between elements)
Think of it like a picture frame: content is the picture, padding is the mat, border is the frame, and margin is the space on the wall!

üíª Activity 2: Padding and Margin

Step 1: Add padding (space inside): 
p {
    padding: 20px;  /* All sides */
    padding: 10px 20px;  /* Top/bottom, left/right */
    padding: 10px 20px 15px 25px;  /* Top, right, bottom, left */
}
Step 2: Add margin (space outside): 
h1 {
    margin: 30px;  /* All sides */
    margin: 20px auto;  /* Top/bottom, left/right (auto centres) */
}

üé® Activity 3: Borders

Step 1: Add borders: 
div {
    border: 2px solid black;
    border: 1px dashed red;
    border: 3px dotted blue;
    border-radius: 10px;  /* Rounded corners */
}

üìä Activity 4: Display Properties

Step 1: Control how elements display: 
/* Block - takes full width, new line */
div {
    display: block;
}

/* Inline - only takes needed width, same line */
span {
    display: inline;
}

/* Flex - flexible layout */
.container {
    display: flex;
}

üìã Complete Example Program

Example: Here's a complete example with layout: 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>CSS Layout Example</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 20px;
            background-color: #f5f5f5;
        }
        
        .container {
            max-width: 800px;
            margin: 0 auto;
            background-color: white;
            padding: 30px;
            border: 2px solid #333;
            border-radius: 10px;
        }
        
        h1 {
            color: #0066cc;
            margin-bottom: 20px;
            padding-bottom: 10px;
            border-bottom: 3px solid #0066cc;
        }
        
        .box {
            background-color: #e0e0e0;
            padding: 20px;
            margin: 15px 0;
            border: 1px solid #999;
            border-radius: 5px;
        }
        
        .highlight {
            background-color: yellow;
            padding: 5px 10px;
            border-left: 4px solid orange;
            margin: 10px 0;
        }
        
        .flex-container {
            display: flex;
            gap: 20px;
            margin-top: 20px;
        }
        
        .flex-item {
            flex: 1;
            background-color: #d0d0d0;
            padding: 15px;
            border-radius: 5px;
        }
    </style>
</head>
<body>
    <div class="container">
        <h1>CSS Layout Example</h1>
        
        <div class="box">
            <p>This box has padding, margin, and a border!</p>
        </div>
        
        <div class="highlight">
            This is a highlighted section with special styling.
        </div>
        
        <div class="flex-container">
            <div class="flex-item">Item 1</div>
            <div class="flex-item">Item 2</div>
            <div class="flex-item">Item 3</div>
        </div>
    </div>
</body>
</html>

üí° Key Concepts

  • Box Model: Content, padding, border, margin
  • Padding: Space inside an element
  • Margin: Space outside an element
  • Border: The edge around an element
  • border-radius: Makes corners rounded
  • display: Controls how elements are displayed
  • flex: A modern way to create flexible layouts
