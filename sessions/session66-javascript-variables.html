// Boolean (true/false)

// Array (list)

    age: 12,
    isStudent: true
};

ğŸ’¡ Key Concepts

  • let: Declares a variable that can be changed
  • const: Declares a constant (can't be changed)
  • String: Text data (in quotes)
  • Number: Numeric data (integers or decimals)
  • Boolean: True or false values
  • Array: A list of items
  • Object: A collection of key-value pairs
  • Template literal: Strings using backticks with ${} for variables
